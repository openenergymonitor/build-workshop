# EmonESP Pixel

**WiFi connected ESP8266**

- Takes serial input string from emonTx e.g. `ct1:30,ct2:103,ct3:56,vrms:40.75`
- Post to [Emoncms.org](https://emoncms.org) via WiFi
- Display visual power value from `ct1` sensor
